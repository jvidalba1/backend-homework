# README

Backend Homework with Movies topic

- Ruby version: 3.1.2
- Rails version: 7.1.3.2

## Installation

- Clone this repository in your local machine
- Go to the folder cloned
- Install dependencies gems
- Setup database
- Create seed data

```bash
git clone <repo_url>
cd <project_folder>
bundle install
rails db:create
rails db:migrate
rails db:seed
```
